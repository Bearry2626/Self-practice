includes -  ตรวจสอบว่า ค่าหนึ่งอยู่ใน Array หรือ String หรือไม่
Ex. String 
const sentence = "This is a delicious soup!";
console.log(sentence.includes("soup"));   true  
console.log(sentence.includes("Soup"));   false เพราะ case sensitive
**includes() จะคืนค่า true ถ้าพบข้อความที่ต้องการใน string
Ex. array
const ingredients = ["onion", "garlic", "bread", "egg"];
console.log(ingredients.includes("garlic"));   true
console.log(ingredients.includes("milk"));     false
**includes() จะคืนค่า true ถ้า element ที่หาอยู่ใน array นั้น
